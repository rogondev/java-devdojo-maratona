# Maratona Java - DevDojo

Este repositório foi criado para documentar toda a minha evolução e aprendizado em Java Core e Orientação a Objetos, seguindo as aulas da "Maratona Java Virado no Jiraya" do DevDojo.

O objetivo principal é construir uma base sólida nos fundamentos da linguagem, entender o gerenciamento de memória por trás do código e evoluir os conceitos de arquitetura e lógica para o desenvolvimento Backend.

---

## 🛠️ Tecnologias e Ferramentas

- **Linguagem:** Java
- **IDE:** IntelliJ IDEA
- **Controle de Versão:** Git / GitHub

---

## 📁 Estrutura do Repositório

O projeto está organizado da seguinte forma para manter o histórico claro e legível:

- `src/`: Contém todos os códigos-fonte divididos em pacotes por assuntos e conceitos estudados (estruturas condicionais, métodos, classes, etc.).
- `anotacoes/`: Arquivos em Markdown detalhando os aprendizados teóricos mais importantes de cada bloco de aulas (como o funcionamento da Stack, Heap e passagem de parâmetros).
